# Chatot - Real-Time Chat Application

A modern, real-time chat application built with the MERN stack, featuring seamless messaging, media sharing, customizable themes, and online status indicators.

**Live Demo:** [https://chatot-4u65.onrender.com/](https://chatot-4u65.onrender.com/)

**GitHub Repository:** [https://github.com/literallyrounak/chatot](https://github.com/literallyrounak/chatot)

---

##  Features

- **Real-Time Messaging**: Instant one-on-one messaging powered by Socket.io
- **Media Sharing**: Share images and other media files directly in chats
- **Profile Pictures**: Customizable user avatars
- **Rich Themes**: Multiple beautiful themes powered by **DaisyUI**
- **Online Status**: Real-time online/offline detection with visual bubbles
- **User Authentication**: Secure signup and login
- **Responsive Design**: Fully optimized for desktop and mobile devices
- **Message History**: Persistent chat history stored in MongoDB

---

## 🛠️ Tech Stack

### Frontend
- **React.js** - UI library
- **Tailwind CSS** - Utility-first styling
- **DaisyUI** - Component library with multiple themes
- **Socket.io Client** - Real-time communication

### Backend
- **Node.js** + **Express.js** - Server framework
- **MongoDB** - NoSQL database
- **Socket.io** - WebSocket implementation for real-time features
- **JWT** - Authentication

### Other Tools
- **Render** - Deployment platform
